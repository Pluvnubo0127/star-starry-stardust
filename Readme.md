游戏名：Star，Starry，Stardust.

类型：文字冒险/指向点击解密游戏

游戏时间：10分钟~（不包含彩蛋）

结局数：1


本游戏需要使用的操作方式为：

鼠标的左键点击，鼠标的左键长按拖拽，

鼠标滚轮的滑动，滚轮动作可以使用方向键的↑↓代替。


本游戏使用的SE（音效）来自：

コニシユカ (nonai sound）：https://metrodelica.booth.pm/

Ｔスタ：https://tnosite.com/

小森平：https://taira-komori.net/freesoundcn.html


本游戏使用的BGM（音乐）来自：

コニシユカ (nonai sound）：https://metrodelica.booth.pm/

にゃるぱかBGM工房：https://nyalpaca.booth.pm/
